##### Usage:
    
    1. generate the expanded map:
        python expand.py # will generate the man4.mat or man6.mat  
        
    2. get the parameter given a point:
        python cspace.py # for example, the function get_param
        
    3. visualization for the current state:
        python vis.py
        
    4. path search
        python path_search.py
        
    5. path following
        python following.py
